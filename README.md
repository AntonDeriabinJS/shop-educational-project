# 🛋️ Интернет-магазин мебели

Учебный проект по вёрстке и фронтенд-разработке.  
Адаптивный многостраничный сайт интернет-магазина мебели с фильтрацией товаров, корзиной и оформлением заказа.

---

## 📄 Страницы

| Страница | Файл | Описание |
|----------|------|----------|
| Главная | `index.html` | Hero-секция, категории мебели, специальные предложения, каталог товаров |
| Каталог | `catalog.html` | Сетка товаров с фильтрацией, сортировкой и хлебными крошками |
| Корзина | `cart.html` | Список товаров в корзине, изменение количества, оформление заказа |

---

## ✨ Реализованные функции

### Общие
- **Адаптивная вёрстка** — три брейкпоинта: десктоп (1200px), планшет (768–1024px), мобильные (&lt; 767px)
- **Единый CSS-файл** — все стили собраны в `style.css` (SCSS → CSS)
- **Hover-эффекты** — плавные переходы (`transition`) на всех кнопках, ссылках и иконках
- **Иконки при наведении** — на карточках товаров появляются иконки корзины и избранного (скрываются на мобильных)
- **Карточки без смещения** — при наведении карточки не увеличиваются и не сдвигаются

### Каталог (`catalog.html`)
- **Фильтрация** — выпадающая панель с фильтрами:
  - Сроки доставки (радиокнопки)
  - Цена (двойной ползунок диапазона)
  - Цвет (14 цветов, чекбоксы с цветными квадратами)
  - Тип комнаты (чекбоксы)
- **Сортировка** — селект: по популярности, цене (возрастание/убывание), новизне
- **Хлебные крошки** — навигационная цепочка
- **Адаптивная сетка** — 3 колонки на десктопе, 2 на планшете, 1 на мобильных

### Корзина (`cart.html`)
- **Список товаров** — изображение, название, описание, цена, количество
- **Изменение количества** — числовое поле ввода
- **Действия** — «Избранные» и «Удалить» (чёрные ссылки с hover-эффектом)
- **Оформление заказа** — форма с полями: имя, телефон, адрес доставки
- **Итоговая сумма** — динамический расчёт
- **Анимация кнопки** — keyframes-анимация появления кнопки «Оформить заказ»

### Футер
- Навигация по сайту
- Контакты (телефон, email)
- Социальные иконки: VK, Telegram, Dzen — при наведении становятся чёрными (`filter: grayscale(100%) brightness(0)`)
- Форма подписки на новости

## 🛠 Технологии

| Технология | Применение |
|------------|------------|
| HTML5 | Семантическая вёрстка, БЭМ-нэйминг |

## 🎨 Дизайн-система

- **Шрифт:** Montserrat
- **Основная ширина контента:** 1200px (центрирование через `padding-left/right: calc(50% - 600px)`)
- **Основные цвета:**
  - Чёрный: `#000000`
  - Тёмно-серый: `#888888`
  - Светло-серый фон: `#F7F6F6`
  - Акцентный белый: `#FFFFFF`
- **Скругление:** 20px (кнопки, поля ввода, карточки)
- **Hover-стиль кнопок:** инвертированные цвета (чёрный фон, белый текст)
